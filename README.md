# Retail Store Revenue Analysis

<p><strong>Practice project</strong> for end-to-end business analytics and machine learning using synthetic retail store data.</p>

<p><mark><strong>Important:</strong> Run <code>python main.py</code> to execute the full pipeline: data generation, data loading, business analysis, chart creation, and machine learning outputs.</mark></p>

<p><mark><strong>Important:</strong> This project uses <code>synthetic data</code> generated by <code>data_generator.py</code>. It is designed for learning, portfolio practice, and pipeline demonstration.</mark></p>

## Overview

This project simulates a local retail business and answers important revenue, product, customer, store, and profitability questions with Python. It also includes machine learning modules for segmentation, prediction, anomaly detection, and basic forecasting.

### What the project does

- Generates retail datasets in `data/`
- Loads, cleans, and merges multiple CSV files
- Solves **30 business questions**
- Creates visualizations in `plots/`
- Runs **4 machine learning tasks**

## Project Workflow

The code follows this pipeline:

1. `main.py` checks whether `data/sales.csv` already exists.
2. If data is missing, `DataGenerator` creates all CSV files.
3. `DataLoader` reads the CSV files and preprocesses them.
4. The merged dataset is passed to `RevenueAnalysis`.
5. `RevenueAnalysis` answers the 30 business questions and saves charts.
6. The same merged dataset is passed to `MLModule`.
7. `MLModule` runs customer segmentation, revenue prediction, anomaly detection, and demand forecasting.

<p><mark><strong>Important:</strong> The main financial logic is based on <code>net_revenue</code> and <code>net_profit</code>, which account for returned transactions.</mark></p>

## Key Business Formulas

These formulas are created inside `data_loader.py`:

- `revenue = price * quantity * (1 - discount_pct / 100)`
- `total_cost = cost * quantity`
- `profit = revenue - total_cost`
- `net_revenue = 0 if is_returned else revenue`
- `net_profit = -total_cost if is_returned else profit`
